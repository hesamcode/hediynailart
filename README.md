# 💅 hediynailart

وب‌سایت رسمی رزرو خدمات ناخن در زعفرانیه تهران.

این پروژه یک لندینگ پیج مدرن و مینیمال برای معرفی خدمات کاشت، ژلیش، ترمیم و طراحی ناخن است که با تمرکز بر تجربه کاربری ساده و رزرو سریع از طریق واتساپ طراحی شده است.

---

## ✨ ویژگی‌ها

- رزرو سریع از طریق واتساپ با متن آماده
- گالری نمونه‌کارها با اسلایدر لمسی
- طراحی ریسپانسیو (مناسب موبایل)
- پشتیبانی از نصب به صورت Web App (PWA)
- ساختار سئو شده (Open Graph، Canonical، Schema)
- بهینه‌سازی برای جستجوی محلی (زعفرانیه تهران)

---

## 📍 موقعیت

تهران، زعفرانیه، بلوار بهزادی  
رزرو فقط با هماهنگی قبلی انجام می‌شود.

---

## 🎯 هدف پروژه

ایجاد یک وب‌سایت سبک، سریع و حرفه‌ای برای افزایش رزرو آنلاین و تقویت برند در فضای دیجیتال.
