## 📖 درباره پروژه

**هنر ناخن هدیه** یک وب‌اپلیکیشن پیشرفته و کاملاً واکنش‌گرا برای رزرو خدمات تخصصی ناخن در زعفرانیه تهران است. این پروژه با معماری **PWA** طراحی شده و امکان نصب بر روی دستگاه‌های مختلف را فراهم می‌کند.

### ✨ ویژگی‌های اصلی

- 🎯 **سیستم رزرو هوشمند** با انتخاب سرویس، تاریخ و ساعت
- 📱 **PWA کامل** با قابلیت نصب بر روی موبایل و دسکتاپ
- 🌙 **تم روشن/تاریک** با ذخیره‌سازی خودکار
- 🌍 **دو زبانه** (فارسی و انگلیسی) با پشتیبانی کامل RTL/LTR
- 🖼️ **گالری تعاملی** با قابلیت سوایپ لمسی
- 📝 **وبلاگ تخصصی** با مقالات آموزشی و TOC هوشمند
- 🔗 **اشتراک‌گذاری** در شبکه‌های اجتماعی
- 📊 **سئوی کامل** با Schema.org و متاتگ‌های استاندارد
- ⚡ **آفلاین** با Service Worker پیشرفته
- 📱 **ریسپانسیو** برای تمامی دستگاه‌ها

## 🗂️ ساختار پروژه

```

hediynailart/
├── assets/
│   ├── css/              # فایل‌های استایل
│   ├── fonts/            # فونت‌های Vazir و Inter
│   ├── images/           # تصاویر گالری، وبلاگ و آیکون‌ها
│   └── js/               # ماژول‌های جاوااسکریپت (۱۲ فایل)
├── blog/                 # وبلاگ فارسی (۴ مقاله)
├── en/                   # نسخه انگلیسی
│   ├── blog/             # وبلاگ انگلیسی (۴ مقاله)
│   └── index.html
├── index.html            # صفحه اصلی
├── manifest.webmanifest  # PWA Manifest
├── sw.js                 # Service Worker
├── robots.txt            # راهنمای موتورهای جستجو
└── sitemap.xml           # نقشه سایت

```

## 🚀 تکنولوژی‌ها

| تکنولوژی | توضیح |

## 🔧 نصب و راه‌اندازی

```bash
# ۱. کلون کردن پروژه
git clone https://github.com/hesamcode/hediynailart.git
cd hediynailart

# ۲. اجرا با Live Server (VSCode)
# کلیک راست روی index.html → Open with Live Server

# یا با Python
python3 -m http.server 8000

# یا با Node.js
npx serve .
```
