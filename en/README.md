## 📖 About

**hediynailart** is an advanced, fully responsive web application for booking professional nail services in Zafaraniyeh, Tehran. Built with **PWA** architecture, it can be installed on various devices.

### ✨ Key Features

- 🎯 **Smart Booking System** with service, date, and time selection
- 📱 **Full PWA** with install capability on mobile and desktop
- 🌙 **Light/Dark Theme** with automatic persistence
- 🌍 **Bilingual** (Persian and English) with full RTL/LTR support
- 🖼️ **Interactive Gallery** with touch swipe support
- 📝 **Specialized Blog** with educational articles and smart TOC
- 🔗 **Social Sharing** on major platforms
- 📊 **Complete SEO** with Schema.org and standard meta tags
- ⚡ **Offline Support** with advanced Service Worker
- 📱 **Responsive** for all devices

## 🔧 Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/hesamcode/hediynailart.git
cd hediynailart

# 2. Run with Live Server (VSCode)
# Right-click on index.html → Open with Live Server

# Or with Python
python3 -m http.server 8000

# Or with Node.js
npx serve .
```
